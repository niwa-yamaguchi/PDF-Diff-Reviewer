# PDF差分レビューア（PDF Diff Reviewer）

改訂前後のPDF図面を読み込み、変更箇所を色分けしてオーバーレイ表示するブラウザ内ツールです。設計変更レビューで大量の図面を目視照合する負担・見落としを減らすことを目的としています。

## 特徴

- **ラスタライズ→画素比較方式**でテキストレイヤに依存せず、ベクタPDF・スキャンPDFの双方で動作
- 差分を色分け表示：**赤＝削除（旧のみ）／青＝追加（新のみ）／グレー＝共通**
- 処理はすべてブラウザ内で完結し、**ファイルを外部に送信しない**
- ページ送り、ズーム／パン、位置合わせ、しきい値・DPI調整、PNG／PDF出力に対応

## 使い方

1. `index.html` をブラウザで開く（`file://` でも動作、サーバ不要）
2. 改訂前・改訂後のPDFをそれぞれ読み込む
3. 差分が色分け表示されるので、ページを切り替えながら確認する

## 依存

- [pdf.js](https://pdfjs.express/) 3.11.174
- [jsPDF](https://github.com/parallax/jsPDF) 2.5.1

いずれも cdnjs から `<script>` で読み込みます（初回表示時はネット接続が必要）。

## デプロイ

`index.html` を含むフォルダを任意の静的ホストに配置するだけです（Cloudflare Pages の Direct Upload、GitHub Pages など）。
